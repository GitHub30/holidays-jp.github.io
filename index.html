Holidays JP API

日本の国民の祝日の一覧を、JSON形式で返すAPIです。

データは Google が用意している Googleカレンダーの情報から自動で生成しています。

去年・今年・来年のデータが取得可能です。

基本API

URL

https://holidays-jp.github.io/api/index.json

返却例

{
  "1420038000": {
      "date": "2015-01-01",
      "title": "元日"
  },
  "1420988400": {
      "date": "2015-01-12",
      "title": "成人の日"
  },
  "1423580400": {
      "date": "2015-02-11",
      "title": "建国記念の日"
  }
}

unixtimeをキーに、date と title が入ったオブジェクトを返します。

年別API

URL

https://holidays-jp.github.io/api/year/2017.json

上記URL 2017の部分に、年号を4桁で指定します。

返却例

{
  "1483196400": {
      "date": "2017-01-01",
      "title": "元日"
  },
  "1483282800": {
      "date": "2017-01-02",
      "title": "元日 振替休日"
  },
  "1483887600": {
      "date": "2017-01-09",
      "title": "成人の日"
  }
}

指定された年のデータだけを返します。返却形式は基本APIと同じです。
